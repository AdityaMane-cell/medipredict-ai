# Smart Health AI
Final-year project – AI-powered disease prediction, diagnosis & preventive first-aid.

## Features
- Multi-label disease prediction from symptoms (XGBoost)
- Real-time first-aid recommendations
- User history & PDF reports (add soon)
- Responsive UI

## Tech Stack
- ML: XGBoost, scikit-learn, pickle
- Backend: FastAPI
- Frontend: React
- Deployment: Render / Vercel
